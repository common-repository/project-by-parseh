== Description ==

<h4>Projects by Parseh</h4>
Display latest projects in page or post with shortcodes

<h4>Featured</h4>
<ul>
<li>A shortcode to show projects list</li>
<li>Widget to show recent projects</li>
<li>Filterable projects list ( with Taxonomy )</li>
<li>An option page to customize the plugin section</li>
</ul>

<h4>Description</h4>
You can create your portfolio's page and display your projects thanks to the portfolio shortcode, 
you can also show your projects as an image gallery selecting the categories that you want to show. 
<strong>Projects by Parseh</strong> works with all wordpress themes.

<h4>Idea or Suggestion ?</h4>
Send us your feedback and suggestions. [projects@parseads.com](mailto:projects@parseads.com)

== Installation ==

1. Upload 'projects-by-parseh.zip' to the '/wp-content/plugins/' directory
2. After extracting , Activate the plugin through the 'Plugins' menu in WordPress
3. Place shortcode '[projects]' in your templates
